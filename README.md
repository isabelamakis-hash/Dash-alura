# 📈 Dashboard de Análise de Dados — Imersão Alura

Dashboard interativo de análise de dados desenvolvido com Python durante a Imersão em Dados da Alura.

🔗 **[Acesse o projeto ao vivo](https://dash-alura.streamlit.app/)**

## 📌 Sobre o projeto

Este projeto foi desenvolvido durante a **Imersão em Dados com Python da Alura**. O objetivo foi construir um dashboard interativo que permite visualizar e explorar dados de forma clara e acessível, utilizando Python e Streamlit.

## 🛠️ Tecnologias utilizadas

- Python 3
- Pandas — manipulação e tratamento de dados
- Streamlit — criação do dashboard interativo
- Plotly / Matplotlib — visualização de dados

## 🚀 Como executar localmente

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/Dash-alura

# Acesse a pasta
cd Dash-alura

# Instale as dependências
pip install -r requirements.txt

# Execute o dashboard
streamlit run app.py
```

Ou acesse diretamente: [https://dash-alura.streamlit.app/](https://dash-alura.streamlit.app/)

## 💡 Funcionalidades

- Visualização interativa de dados
- Filtros dinâmicos
- Gráficos e métricas em tempo real
- Interface intuitiva via Streamlit
